# NFHC
この拡張機能はPixivとカクヨムのルビなど、特殊な独自構文をハイライトしそこに補完機能も付け合わせたものです。<br>
This extension highlights a special proprietary syntax such as Pixiv and ruby of Kakuyom and adds complementary functions there.<br>
